# jsdoc README

gen jsdoc

## How to use

1. Select the parameters of a function
2. Press `ctrl+shift+p`
3. Type `gen jsdoc`
4. Choose `gen jsdoc`

## Requirements

none

## Extension Settings

This extension contributes the following settings:

- `jsdoc.author`: Add this option to Author be inserted automatically
